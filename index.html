getting help

Command Description
git help git help page
git help [command] look up documentation on [command]
git help | less pipe git help into less

configuration

Command Description
git config --global user.name "Your Name" set Git username
git config --global user.email your.email@example.com set Git email address
Git stores global configuration settings in a hidden text file located in your home directory (~/.gitconfig).

viewing commits

Command Description
git log view log of commits
code log -p view log of commits with details
git diff --staged diff previous version of repo with staged changes
git diff branch-1 branch-2 diff branch-1 and branch-2
git show <SHA> show diff vs <SHA>

branches

Command Description
git checkout <BRANCH> check out <BRANCH>
git checkout -b <BRANCH> create and checkout <BRANCH>
git branch list all local branches
git branch -a list all (including remote) branches
git merge about-page merge about-page branch with current branch (i.e. from master)
git branch -d <BRANCH> delete <BRANCH> (if merged)
git branch -D <BRANCH> delete branch (even if unmerged) (dangerous)
git checkout -f reset all changes and revert back to latest commit (HEAD)
git reset --hard HEAD same as previous

remote

Command Description
git remote add origin <URL> add remote connection at <URL>
git push -u origin master push to master branch of upstream repo
git push push to default remote
git clone <clone URL> project-name clone repo at <clone URL>. project-name is optional
git pull pull in changes from remote repository
